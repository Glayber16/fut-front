Front-end de um projeto pessoal sobre ida ao estadio

O projeto permite:

Salvar quais partidas eu fui

Registrar com quem fui

Informar data e placar

Visualizar uma página de estatísticas com os resultados geral

Tecnologias utilizadas no projeto:

![Next.js](https://img.shields.io/badge/Next.js-14-black?style=for-the-badge&logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

shadcn/ui para componentes reutilizáveis

lucide-react-icons para ícones

Prettier para manter o código limpo e padronizado
